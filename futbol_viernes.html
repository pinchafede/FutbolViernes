SETUP Y CONFIGURACIÓN
Cómo conectar Google Sheets y publicar en GitHub Pages
💡

Ahora mismo la app corre con datos de ejemplo. Seguí estos pasos para conectar tu Google Sheet real y publicar la app gratis en GitHub Pages.

1

Crear el Google Sheet

Creá una planilla nueva en Google Sheets con 3 solapas con exactamente estos nombres:

📋 Hoja 1: "jugadores"
   Columnas: nombre | apodo | color

📋 Hoja 2: "partidos"
   Columnas: fecha | equipo_a | equipo_b | goles_a | goles_b | mvp | notas

📋 Hoja 3: "stats"
   Columnas: fecha | jugador | goles | asistencias | equipo

La fila 1 de cada hoja son los encabezados (como están arriba). Los datos arrancan en la fila 2.

2

Hacer la planilla pública (solo lectura)

En Google Sheets → Compartir → "Cualquier persona con el enlace puede ver". Esto permite que la app lea los datos sin necesidad de login.

3

Obtener el ID de la planilla

Abrí la planilla y copiá el ID de la URL:

https://docs.google.com/spreadsheets/d/ESTE_ES_EL_ID/edit
4

Obtener una API Key de Google

1. Entrá a console.cloud.google.com
2. Creá un proyecto nuevo (o usá uno existente)
3. Activá la Google Sheets API
4. Credenciales → Crear credenciales → Clave de API
5. Copiá la clave generada

5

Configurar la app

Abrí el archivo config.js y completá tus datos:

const CONFIG = {
  SHEET_ID: "tu_id_de_planilla_aqui",
  API_KEY:  "tu_api_key_aqui",
  SEASON:   "2025"
};
6

Publicar en GitHub Pages

1. Creá un repo en GitHub (puede ser privado)
2. Subí los archivos index.html y config.js
3. Settings → Pages → Branch: main → Save
4. En 1-2 minutos tu app está en: tuusuario.github.io/futbol-viernes

7

Cargar datos cada viernes

Después de cada partido, el admin agrega una fila en la hoja stats (un jugador por fila) y actualiza la hoja partidos con el resultado. La app se actualiza automáticamente.
